
# Go-crud

Go-curd is a CRUD API for a blog application written using Golang.





## Tech Stack

**Server:** Golang, Gin-Gonic, GORM

**Database:** PostgreSQL




## Environment Variables

All the environment Variables are already mentioned in .env file.
